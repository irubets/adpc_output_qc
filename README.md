QC the adnca dataset (XPT, SAS7BDAT, or CSV) and produce the report in HTML format and Excel file with the findings. Requires full path to the location of the following items: (1) ADNCA dataset, (2) the adnca.sas program, (3) the Specifications.xlsx file, (4) SAS log file. Code reads all necessary paths and some configuration mapping data from adnca-qc.cfg file, which you can create yourself using a template.cfg file, or it can create it for you. IMPORTANT: you have to provide the FULL path to each item, and if the path is long and contains spaces, include it in double quotes. If you have already create the adnca-qc.cfg yourself, submit it as follows: python adpc_output_qc_run.py --input "path\to\adnca-qc.cfg". Otherwise just run python adpc_output_qc_run.py.
